node js - это компилятор js на компьютере

npm init - инициализация проекта

Для запуска конкретного файла - node "название файла"
Для запуска проекта - в package.json в объекте scripts написать ключ(название) и команду, которая будет выполняться
И в терминале пишем npm run "ключ"
Для установки пакета(библиотека) - npm install(i) "название пакета"

В package.json указать "type": "module" для использования import
